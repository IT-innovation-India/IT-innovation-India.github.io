Cloud Computing with AWS

Learning Objective

To learn various technology and business aspects of the cloud and to implement application on cloud

Achievement

  • To understand concepts and terminologies of Cloud Computing
  • To understand business perspectives of Cloud Computing
  • To understand technical perspectives - Virtualization, CDN (Content Delivery Network), Cloud Storage, Networking
  • To be able to adopt Cloud Computing in the enterprise
  • To be able to cope with the main challenges of Cloud Computing including Governance, Risk and Security
  • To be able to effectively practice Cloud Architecture and Cloud Development
  • To understand the implementation process of cloud in enterprises using hands-on-labs and projects

Learner Profile

Working experience –3-8 years | Basic knowledge of Internet and Web programming desirable Communication capability – MED

Course Overview

Duration 2 weeks Hours
Cloud Computing - Introduction, Adoption, Governance and Security, Architecture/ Development Practices 28
Cloud Service Exercises (AWS) 21
