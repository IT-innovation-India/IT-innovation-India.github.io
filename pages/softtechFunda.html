Software Technology Fundamentals

Learning Objective

To learn the core areas of software engineering and development life cycle including architecture, data structures, database, programming languages and quality

Achievement

  • To be able to explain key aspects of models and processes for designing a complex software system
  • To be able to apply key aspects of a software engineering process to the development of a complex software system
  • To be able to evaluate requirements for a complex software system
  • To be able to apply the process of analysis and design using the object-oriented approach

Learner Profile

Working experience – 0-3 years | Knowledge of structured programming concepts | Fluency in the basics of the software development process | Basic knowledge in C or Pascal Communication capability – LOW-MED
